## Our uni Project

This project uses C and python for the logic, JavaScript as the midleman, and HTML as the display.

### Setup
1. run 'make'
2. run 'make html' to open the html without re-compiling stuff

### Technologies
- C
- python
- JavaScript
- HTML/CSS

### dependancys
this project uses the emcripten compiler-toolchain to compile c into web assembly
